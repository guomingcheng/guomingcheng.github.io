# 英语的五种基本句式

学习在英语中几种词语组合写作方式.


 第一种基本句式: 主语 + 谓语(动词)
 第二种基本句式: 主语 + 系动词 + 表语
 第三种基本句式: 主语 + 谓语(动词) + 宾语
 第四种基本句式: 主语 + 谓语(动词) + 间宾 + 值宾
 第五种基本句式: 主语 + 谓语(动词) + 宾语 + 宾补

# 分别是

主语 > ( 句子陈述的对象, 这个句子是围绕谁展开，谁就是主语 )
谓语 > ( 描述主语接下来发生的动作，用来说明主语怎么了，就用动词来表达，这就是为谓语 )
表语 > ( 又可以叫做主语补足语，是补充主语这个对象更为具体，紧跟在系动词后面的语句 ) 

# 主语
如: I study English . 我学英语
释: I - 就是主语, 这条语句陈述的对象就是我来展开

# 谓语
如: I drink water . 我喝水
释: I - 就是主语, drink - 就是动词。I 发出 drink 这个动作

宾语 > ( 是主语动作承受的对象、宾语是描述了一个具体的东西，也常常是一个名词 )
系动词 > ( 又可以叫做系动词、这词是没有具体的动作，只是起到连接主语和后面其他语句的作用 ) 

如: be(am， is, are， was, were) look/feel/sound/smell/taste/seem/become/turn/grow/get/gp/fall

双重宾语 ( 包含了直接宾语和间接宾语 直接宾语, 是动作的承受者对象 )
