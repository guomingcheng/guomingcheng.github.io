# 简介

#
vue 是一个 js 框架，对数据处理有绝对的优势

vue 是一种用于构建用户界面数据的渐进式框架，它的核心是对于数据的处理。

vue 通过绑定或者说挂载父元素，对其子元素进行数据驱动，当要改变元素的数据时，我们是不需要去操作 dom 的，只需要把 data 的数据修改，而 dom 元素引用的数据也会同步的修改的m

# 第一个 Vue 的实列

#
认识一下 Vue 的元素 le 挂载

le 挂载规则以及作用范围描述：

1、 Vue 会管理 le 挂载命中的元素以及它的内部元素
2、 le 挂载，不局限于 ID 选择器，其他的也是可以的。不过建议使用 ID 选择器，因为它是唯一的
3、 le 挂载不能用于 html body 这俩个元素，其余的都可以

<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <title>Title</title>
   <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
</head>
<body>

   <div id="app">
       {{message}}
   </div>

</body>

<script>

   var vue = new Vue({
       el: '#app',
       data:{
           message: 'dddd'
       }
   })
</script>

</html>

#
认识一下 Vue 的 data 数据驱动

data 的数据驱动不止简单类型，对象类型，数组类型的都可以。

Vue 的数据模型都是定义在 data 中，而宣染复杂数据模型时，遵守 js 语法即可

<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <title>Title</title>
   <script src="https://cdn.jsdelivr.net/npm/vue/dist/vue.js"></script>
</head>
<body>

   <div id="app">
       {{message}}
       <h1>{{bat.id}} : {{bat.name}}</h1>
       <span>{{vu[0]}}</span>
   </div>

</body>

<script>

   var vue = new Vue({
       el: '#app',
       data:{
           message: '郭明城',
           bat: { id: "1", name:"礼言"},
           vu:[1, 2, 3]
       }
   })
</script>

# 内容绑定

#
v-text 指令的使用与实列

v-text 指令也是内容绑定

，但与 "{{}}"语法有一点不同，指令是把元素的内容全部删除在填充

<body>
   <div id="app">
       <h2 v-text="message">你好：</h2>
       <h2 v-text="message + !!!!">你好：</h2> // 这里的 + 运算符就是字符串拼接
   </div>
</body>

<script>
   var vue = new Vue({
       el: '#app',
       data:{
           message: '郭明城',
       }
   })         输出： 郭明城
</script>     输出： 郭明城!!!!

#
v-html 指令的使用与实列

v-html 是向父元素添加子元素

它与 v-text 的区别在于，如果内容中有 html 结构，它会解析成标签，而 v-text 指令无论内容是什么，只会解析文本

<body>
   <div id="app">
       <p v-html="message"></p>
   </div>
</body>

<script>
   var vue = new Vue({
       el: '#app',
       data:{
           message: '<a> 你在那一边吗</a>',
       }
   })
</script> 输出：标签结构为， <p><a></a></p>

#
v-on 指令的使用与实列

v-on 指令可以监听任何的监听事件

监听事件都是写在 v-on: 后面，而我们的监听方法需要在 methods 属性中声明，方法中操作的 data 属性的数据，是可以刷新页面元素的数据

<body>
   <div id="app">
       <input type="button" value="点击事件" v-on:click="funclick">
       <p v-text="message"></p>
   </div>
</body>

<script>
   var vue = new Vue({
       el: '#app',
       data:{
           message: '你在那一边吗',
       },
       methods:{
           funclick: function (){
              this.message = "你去哪了"
           }
       }
   })
</script>

# 显示切换

#
v-show v-if 指令的隐藏与显示

只要用于显示或者隐藏某个元素

v-show 的值可是真与假, 也可以是表达式, 而表示式是被 Vue 转换成布尔值的。

v-show 与 v-if 的不同在于，v-if 的隐藏是把元素直接删除，而 v-show 则是添加 display: none 属性把当前的标签隐藏掉

<body>
   <div id="app">
       <p v-show="message">测试 v-show 指令</p>
       <p v-show="age >= 18">测试 v-show 指令的表达式</p>
   </div>
</body>

<script>
   var vue = new Vue({
       el: '#app',
       data:{
           message: false,
           age: 17
       },

   })
</script>   输出：v-show 的结果都为假，那么它俩个元素是不会显示的

#
v-bind 的绑定属性功能

在 html 中，元素是有很多属性的，而 v-bind 作用就是为属性设值

v-bind 指令的写法就如同下面，它可以为所有的属性都能绑值，不仅仅是 src, 属性如：class 等等的多种属性.....

<body>

   <div id="app">
       <img v-bind:src="imgSrc"></img>
   </div>

</body>

<script>

   var vue = new Vue({
       el: "#app",
       data: {
           imgSrc: "img/仓库定位.png"
       }
   })
</script>
