# 简介

如果要构建自己的 dapp 应用，就需要使用官方整合的 web3.0.js 库，用于调用 abi 二机制接口，这样才能实现与以太坊节点通信，才能构建 dapp 应用

# solc.js

这是一款编译器, 我们如果在本地编写 .sol 智能合约, 就需要使用 solc 编译器，把 sol 文件编译成二机制机器码与 abi 二进制接口

#
安装 solc

注意：在安装之前需要先安装 ndeo, 因为需要使用到 npm 命令。

注意：命令使用时，不是 solc, 而是 solcjs

# web3.0

web3.js 是一个 js 库, 他可以调用任何暴露出 RPC 层以太坊节点 abi 接口, 使的 dapp 应用与以太坊通信. web3.js 还内置了各种实用对象，其中就包含了 web.eth 对象，用于获得区块链的数据。

#
安装 web3.0

版本查看命令：npm list web3

# 二机制接口（abi）

通过 web3.js 来调用以太坊智能合约的 abi 接口, 需当把智能合约编译成 json 数据, 用来在 js 代码中描述, 在这数据中每个对象都对应着一个共有的方法, 这个对象的属性分别是以下作用。

#
js 描述一个 abi 接口

{
   "constant" : true; //函数内容操作氛围, constant 常量, true 表示当前函数是只读的, 不会修改状态变量
   "inputs" : [        //表示函数的传参描述
       {
           "name": "", // 参数的名称
           "type": "address", //参数的类型    
       }
   ],
   "name":"mint", // 函数的名称
   "outputs": [        //表示函数的返回参数描述
       {
            "name": "", // 参数的名称
            "type": "address", //参数的类型    
       }
    ],
   "payable": false, //表示当前函数不是 payable 类型的
   "stateMuttability": "view" //函数的内容操作范围
   "type": "function"  //函数的类型，取值分别是普通函数 function, 构造函数 constructor , 事件函数 event 

}
