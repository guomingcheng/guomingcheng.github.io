# tblib 优势说明

# telegramTblib 是官方提供可定制化得 Telegram, 让开发者可以对 Telegram 进行二次开发

这个 tblib 想相当于 bot 机器人的加强版, 因为机器人是无法向用户主动发送信息，只有当用户信息出现，机器人才能做出响应发送。 而 tblib 就相当用户使用 api 来操作这个 telegram 软件, 官方也为开发者提供了所有 api 权限操作接口 。 当我们使用 tg 软件，是可以看到所有群组和群组的信息，加群，退群，加人，删人等操作。因此，我们使用 tblib 二进制接口也能实现的了
