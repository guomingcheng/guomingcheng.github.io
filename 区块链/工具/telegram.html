# telegramBot 说明

bot 机器人是一个群助手补助工具，管理群的设置，并可以监听群里的信息内容，并进行回复: 机器人是可以回复各种类型的信息，可以文字、图片、视频、投票、以及复杂的交互。

# 在群、或者频道里。机器人是可以主动的推送消息

虽然机器人有着强大的功能, 但是只能触发是回复，无法主动的向用户发送信息, 这也许是开发者考虑的机器人被滥用的原因吧

#  通信方式:

#     1. webhook
webhook 是与机器人通信最有效率的方式, 架设一个 web 机器人服务与机器人绑定在一起。每当 telegramBot 
收到新的消息就会向这个地址传递信息的内容。

使用这个 API 绑定 webhook 服务
https://api.telegram.org/bot5476727522:AAF7QK0-OygyrKTeHPsH8U0ajInX2rkIOHs/setWebhook?url=
#     2. 轮询
轮询的方式是每次都要向服务器询问是否有新的消息，所有也没有什么效率。但是编写代码就比较简单一些
